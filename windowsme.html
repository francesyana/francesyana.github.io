Windows ME Installation

This section provides a step-by-step demonstration of installing Windows ME. Watch the instructional video below to follow along with the process.

Product Keys

  • B6BYC-6T7C3-4PXRW-2XKWB-GYV33
  • FMMMC-G8GQH-BH9MT-YBRH3-CRD6T
  • P8DJW-RQXXM-T8JCW-HFB29-4FK3Y
  • KKK3F-QKXGY-WHG99-B92WK-GFK4C

Steps

  1. Create a new VM.
  2. Set Base Memory to 256 MB.
  3. Add a bootable diskette if your ISO is not bootable in the floppy disk.
  4. Add the ISO file.
  5. Start the VM.
  6. Choose 2 → then 1 → Enter → Enter → Yes → Enter → choose 2 again → then 1 → wait for Drive C to be formatted → Enter to continue.
  7. In Setup: Next → Next → wait → Typical → Next → Next → install the most common → Computer Name: EndUser → Country: Philippines → Next → Time zone: +8 → Next → Finish → wait → Restart now.
  8. Select 1 → User Info: EndUser (same for company) → I accept → Product Key: GKDMG-XV6JQ-BVDB7-WQ7YY-VTQRD → Next → Finish → Restart.
  9. Choose 1 → wait → Restart.
  10. Choose 1 → wait → Admin & password → confirm password → OK → wait.
