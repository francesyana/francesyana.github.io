Active Directory: Creating Users

This section presents mass user creation in Windows Server 2008 via PowerShell scripts and text files.

Requirements

  1. Create user accounts for all BSIT students (1st to 4th year) with a common password.
  2. Group the students according to their section.
  3. Your account belongs to Administrators.
  4. Domain: bsit.com
  5. Common password: Student12345

Steps

  1. Prepare a text file containing student names grouped by section.
  2. Open Windows Server 2008 → Start → Administrative Tools → Active Directory Users and Computers.
  3. Create Organizational Units (OUs) for each year level and section.
  4. Use PowerShell to automate user creation:
    • Import the text file with student names.
    • Run a script to create accounts under the domain bsit.com.
    • Assign the common password Student12345 to all accounts.
  5. Verify that each student account is created and grouped correctly.
  6. Ensure your administrator account is part of the Administrators group.
