Case Study Part 5

This section presents Case Study Part 5: RAID Installation.

Steps

  1. Clone the Windows Server 2008 VM used in the DHCP activity.
  2. Open Settings → Storage → click the add attachment icon → Hard Disk.
  3. In the medium selector, click Create → check pre-allocate full size → allocate 50 MB → Finish.
  4. Create 10 of these hard disks.
  5. Back in Settings → Storage → attach all 10 hard disks to the WS2008 VM.
  6. Start WS2008.
  7. Click Start → Administrative Tools → Server Manager → expand Storage → Disk Management.
  8. When prompted, initialize all disks with MBR partition style → click OK.
  9. RAID 0: Striped Volume
    • Select Disk 1 → New Striped Volume.
    • Add Disk 2 → Next → Next.
    • Name volume: RAID0 (Striped).
    • Check “Perform a quick format” → Next → Finish → confirm prompt → wait.
  10. RAID 1: Mirrored Volume
    • Select Disk 3 → New Mirrored Volume.
    • Add Disk 4 → Next → Next.
    • Name volume: RAID1 (Mirrored).
    • Check “Perform a quick format” → Next → Finish → confirm prompt → wait.
  11. Spanned Volume
    • Select Disk 5 → New Spanned Volume.
    • Add Disk 6 and Disk 7 → Next → Next.
    • Name volume: Spanned.
    • Check “Perform a quick format” → Next → Finish → confirm prompt → wait.
  12. RAID 5
    • Select Disk 8 → New RAID-5 Volume.
    • Add Disk 9 and Disk 10 → Next → Next.
    • Name volume: RAID5.
    • Check “Perform a quick format” → Next → Finish → confirm prompt → wait.
