Case Study Part 9

This section presents Case Study Part 9: Windows Deployment Services (WDS). In this activity, we add a Windows 7 image to WDS running on Windows Server 2008.

Steps

  1. Prepare the ISO
    • Mount the Windows 7 ISO in your WS2008 VM.
    • Open Computer → right‑click the DVD drive → Open.
    • Copy all files from the DVD drive into a local folder: C:\WDSImages\Win7.
  2. Add the Install Image
    • Open Windows Deployment Services from Administrative Tools.
    • Expand your server → right‑click Install Images → Add Install Image.
    • Create a new image group: WIN7-AIO.
    • Browse to: C:\WDSImages\Win7\sources\install.wim.
    • Click Next → wait for import → Finish.
  3. Add the Boot Image
    • Right‑click Boot Images → Add Boot Image.
    • Browse to: C:\WDSImages\Win7\sources\boot.wim.
    • Name it: Win7 Boot.
    • Finish the wizard.
  4. Deploy Windows 7
    • Create a new VM in VirtualBox for Windows 7.
    • Set Network Boot (PXE) as the first boot option.
    • Start the VM → press F12 when prompted.
    • WDS will load → enter credentials (Username and Password).
    • Select Windows 7 Ultimate → Next → Install.
