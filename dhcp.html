Activity 3 & 4

This section presents Configuring DHCP Server for Windows Server 2008 and connecting it to Windows XP & Windows 7.

Scopes

  • Male network: 10.0.0.0 /8
  • Female network: 172.16.0.0 /16
  • Server: Last usable IP address
  • Exclude: First & last 10 usable IP addresses
  • Lease Time: 16 hours
  • Router: Server IP address
  • DNS: Server IP address (8.8.8.8)

Requirements

  1. Create client OS: 1 Windows XP (LASTNAME-XP) and 1 Windows 7 (LASTNAME-WIN7).
  2. Connect both client OS to the Windows Server Domain using DHCP.
  3. Windows 7 (Male network) → 10.0.0.0 /8 → Class B IP addresses only.
  4. Windows XP (Female network) → 172.16.0.0 /16 → Class A IP addresses only.
  5. Login 5 student accounts per section per OS.

Steps

  1. Clone Windows Server 2008.
  2. Clone Windows 7 and Windows XP.
  3. Configure WS2008 network adapters: Adapter 1 & Adapter 2 → Internal Network.
  4. Configure Windows 7 and XP network adapters: Adapter 1 → Internal Network.
  5. Start WS2008.
  6. Configure Networking:
    • (Male) Local Area Connection 1: IP 10.0.0.254, Subnet 255.255.255.0, Gateway 10.0.0.1, DNS 127.0.0.1.
    • (Female) Local Area Connection 2: IP 172.16.0.126, Subnet 255.255.255.128, Gateway 0.0.0.0, DNS 127.0.0.1.
  7. Add DHCP Role: Start → Administrative Tools → Server Manager → Roles → Add Roles → DHCP Server. Validate DNS (127.0.0.1) → Alternate DNS: 8.8.8.8 → Install.
  8. Configure DHCP Scopes:
    • Scope Name: Class A Male
      • Range: 10.0.0.1 – 10.0.0.254
      • Subnet mask: 255.255.255.0
      • Exclusions: 10.0.0.1–10.0.0.10 and 10.0.0.245–10.0.0.254
      • Lease: 16 hours
      • Default gateway: 10.0.0.254
      • Domain: bsit.com
    • Scope Name: Class B Female
      • Range: 172.16.0.1 – 172.16.0.126
      • Subnet mask: 255.255.255.128
      • Exclusions: 172.16.0.1–172.16.0.10 and 172.16.0.117–172.16.0.126
      • Lease: 16 hours
      • Default gateway: 172.16.0.126
      • Domain: bsit.com
  9. Configure DNS:
    • Class A: Default gateway 10.0.0.254, Alternate DNS 8.8.8.8.
    • Class B: Default gateway 172.16.0.126, Alternate DNS 8.8.8.8.
  10. Configure Windows XP:
    • Rename computer to LASTNAME-XP.
    • Join domain: bsit.com.
    • Authenticate with WS2008 Administrator credentials.
    • Restart and log in via domain.
  11. Configure Windows 7:
    • Rename computer to LASTNAME-WIN7.
    • Join domain: bsit.com.
    • Authenticate with WS2008 Administrator credentials.
    • Restart and log in via domain.
  12. Verify DHCP leases:
    • WS2008 → Administrative Tools → DHCP → IPv4 → Address Leases.
    • Check W7 lease in Class A Male scope.
    • Check WXP lease in Class B Female scope.
    • Run ipconfig /all in W7 and WXP to confirm DHCP assignment.
  13. Login 5 student accounts per section per OS.
