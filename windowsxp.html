Windows XP Installation

This section provides a step-by-step demonstration of installing Windows XP. Watch the instructional video below to follow along with the process.

About Windows XP

Windows XP is a major version of Microsoft's Windows NT operating system, released in October 2001. It combined the stability of the NT kernel with a user-friendly interface, becoming one of Microsoft’s most successful, stable, and popular consumer and business operating systems, lasting until its mainstream support ended in 2009.

Steps

  1. Open VirtualBox and create a new VM.
  2. Select Windows XP as the version.
  3. Allocate RAM.
  4. Create a virtual disk.
  5. Mount the Windows XP ISO file in Storage settings.
  6. Start the VM and boot from the ISO.
  7. At the setup screen, press Enter to install Windows XP.
  8. Accept the license agreement by pressing F8.
  9. Create or select a partition → format using NTFS.
  10. Wait for setup to copy files and restart.
  11. Continue with the graphical setup wizard.
  12. Enter product key, user name, and organization.
  13. Set computer name and administrator password.
  14. Configure regional settings (e.g., Philippines, US Keyboard).
  15. Set date, time, and timezone.
  16. Configure networking (Typical settings recommended).
  17. Allow setup to finish copying files and restart.
  18. Log in to Windows XP for the first time.
