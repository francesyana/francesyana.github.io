Windows 8 Installation

This section provides a step-by-step demonstration of installing Windows 8. Watch the instructional video below to follow along with the process.

Windows 8 Installation Steps

  1. Create a new VM.
  2. Select Windows 8 (32-bit).
  3. Add ISO file.
  4. Start VM.
  5. Select language to install → Continue → Install now.
  6. Enter product key.
  7. Accept license agreement.
  8. Choose Custom Install Windows → Next → wait for installation.
  9. Personalize → PC Name: Admin-PC → Customize.
  10. Choose “No, don’t turn on” for settings → disable Windows Update → turn off everything → Next.
  11. Continue setup → turn off everything in each prompt → Next.
  12. Create local account → fill in info to sign in.
