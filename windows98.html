Windows 98 Installation

This section provides a step-by-step demonstration of installing Windows 98. Watch the instructional video below to follow along with the process. Windows 98 Installation Steps; 1. Create your VM for Windows 98 and add your ISO file. 2. Go to Settings → Storage 3. In attributes, at the right, in optical drive, select your iso then select IDE Primary Device 1. 4. Start VM. 5. In startup menu, select 2 (Boot from CD ROM) → select 1 → wait. 6. Now in Setup, press Enter to continue → enter again → enter → select Yes → enter → enter. 7. Now you're back in Startup Menu → Select 2 again. Then, select 1. 8. Wait for the hard disk to be completely formatted. 9. After that, select continue → wait. 10. A Select Directory tab should pop up and click Next → wait. 11. In Setup options, select Custom → next → next. 12. Name your computer (BSIT) → next → select Keyboard Layout → next → location : Philippines → next → next → wait 13. Restart Now → Boot from Hard Disk → name: End User → Company: End User → I accept → Product Key→ DMWRC - T9XP4 - GJ2P8 - 26G66 - V9WBB → next → finish → wait. 14. If there is an error, restart machine then hold left ctrl → dont select anything until time runs out → then select safe mode → wait → click ok. 15. Go to computer → Properties → performance → click file system → troubleshooting → disable 32 bit protectod mode disk driver → apply → close → close → the restart computer. 16. Select 1 → wait → End User then password → Timezone: Beijing → Apply → ok → wait → restart → select 1 → Log in → Installed win 98.
