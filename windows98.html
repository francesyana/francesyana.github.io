Windows 98 Installation

This section provides a step-by-step demonstration of installing Windows 98. Watch the instructional video below to follow along with the process.

Resources

  • Windows 98 SE ISO: Download here

About Windows 98

Windows 98 is a consumer-oriented graphical operating system released by Microsoft on June 25, 1998, as the successor to Windows 95. It is part of the Windows 9x family, utilizing a hybrid 16-bit/32-bit architecture based on MS-DOS. It was known for introducing improved USB support, FAT32 file system, and web integration (Internet Explorer) into the desktop interface.

Steps

  1. Create your VM for Windows 98 and add your ISO file.
  2. Go to Settings → Storage.
  3. In attributes, at the right, in optical drive, select your ISO then select IDE Primary Device 1.
  4. Start VM.
  5. In startup menu, select 2 (Boot from CD ROM) → select 1 → wait.
  6. In Setup, press Enter to continue → Enter again → Enter → select Yes → Enter → Enter.
  7. Back in Startup Menu → Select 2 again → then select 1.
  8. Wait for the hard disk to be completely formatted.
  9. Select Continue → wait.
  10. In Select Directory tab, click Next → wait.
  11. In Setup options, select Custom → Next → Next.
  12. Name your computer (BSIT) → Next → select Keyboard Layout → Next → Location: Philippines → Next → wait.
  13. Restart Now → Boot from Hard Disk → Name: End User → Company: End User → I accept → Product Key: DMWRC-T9XP4-GJ2P8-26G66-V9WBB → Next → Finish → wait.
  14. If error occurs: restart → hold Left Ctrl → let timer run out → select Safe Mode → wait → click OK.
  15. Go to Computer → Properties → Performance → File System → Troubleshooting → disable 32-bit protected mode disk driver → Apply → Close → Restart.
  16. Select 1 → wait → End User login → Password → Timezone: Beijing → Apply → OK → Restart → select 1 → Log in → Installed Windows 98.
