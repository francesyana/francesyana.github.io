Group Policy Management

This section presents Configuring Group Policy Management in Windows Server 2008.

Initial Setup Steps

  1. Go to Start → Administrative Tools → Active Directory Users and Computers.
  2. Create a new Organizational Unit → BSIT.
  3. Add all students to the Organizational Unit.
  4. Go to Start → Administrative Tools → Group Policy Management.
  5. Navigate to Domain → View the OU of Students.
  6. Right-click the OU → Create a GPO in this domain → Add GPOs needed for your system.
  7. After creating the GPO → Right-click the GPO → Edit, then follow the steps below for each requirement.

User Configuration Policies

GPO Name: User Interface Restrictions

  • Remove links and access to Windows Update → Enabled
  • Remove Network Connections from the Start Menu → Enabled
  • Remove Help menu from the Start Menu → Enabled
  • Remove Run menu from the Start Menu → Enabled
  • Remove My Pictures icon from the Start Menu → Enabled
  • Remove My Music icon from the Start Menu → Enabled
  • Remove My Network Places icon from the Start Menu → Enabled
  • Add "Logoff" on the Start Menu → Enabled
  • Remove Task Manager from CTRL+ALT+DEL → Enabled

GPO Name: Internet Explorer Settings

  • Internet Explorer Browser Title – Title Bar Text → <domain name>

GPO Name: Windows Update

  • Prohibit access to properties of components of a LAN connection → Enabled
  • Prevent access to the command prompt → Enabled
  • Prevent access to registry editing tools → Enabled
  • Prohibit access to the Control Panel → Enabled

Computer Configuration Policies

GPO Name: Admin Interface Restrictions

  • Remove links and access to Windows Update → Disabled
  • Remove Network Connections from the Start Menu → Disabled
  • Remove Help menu from the Start Menu → Disabled
  • Remove Run menu from the Start Menu → Disabled
  • Remove My Pictures icon from the Start Menu → Disabled
  • Remove My Music icon from the Start Menu → Disabled
  • Remove My Network Places icon from the Start Menu → Disabled
  • Add "Logoff" on the Start Menu → Disabled
  • Remove Task Manager from CTRL+ALT+DEL → Disabled

GPO Name: Admin Windows Update

  • Prohibit access to properties of components of a LAN connection → Disabled
  • Prevent access to the command prompt → Disabled
  • Prevent access to registry editing tools → Disabled
  • Prohibit access to the Control Panel → Disabled
