Windows 2000 Installation

This section provides a step-by-step demonstration of installing Windows 2000. Watch the instructional video below to follow along with the process.

Product Keys

  • DDTPV-TXMX7-BBGJ9-WGY8K-B9GHM
  • VXKC4-2B3YF-W9MFK-QB3DB-9Y7MB (Alternative)

Steps

  1. Create a new VM.
  2. Set Base Memory to 2048 MB.
  3. Add ISO file.
  4. Start VM.
  5. Press Enter → F8 → Enter → choose NTFS file system → Enter → wait → Enter to restart.
  6. Next → Next → enter name: EndUser → Next → enter product key.
  7. Alternative product key available if needed.
  8. Set Computer name: WIN2000.
  9. Enter Administrator password.
  10. Set Timezone: Beijing → Next.
  11. Choose Custom → Next → Next → Next → wait → Finish.
  12. Wait for restart → Next → type Administrator password → Next.
