Windows 11 Installation

This section provides a step-by-step demonstration of installing Windows 11. Watch the instructional video below to follow along with the process.

Windows 11 Installation Steps

  1. Create a new VM.
  2. Add ISO file.
  3. Start VM.
  4. If boot error occurs: power off → Settings → System → uncheck “Enable UEFI.”
  5. Start VM again.
  6. Next → Next → I agree → Next → “I don’t have product key” → choose Windows 11 Pro → Next.
  7. If error “doesn’t meet requirements” appears → Close → Reset machine.
  8. After restart, press Shift + F10.
  9. Type regedit.
  10. Navigate to HKEY_LOCAL_MACHINE → Setup → right‑click → New → Key → name it LabConfig.
  11. Inside LabConfig: right‑click → New → DWORD (32‑bit) Value → BypassTPMCheck.
  12. Create another DWORD (32‑bit) Value → BypassSecureBootCheck.
  13. Right‑click each → Modify → set Value Data to 1.
  14. Close Registry Editor and Command Prompt.
  15. Proceed again with installation (Step 6).
  16. Select Windows 11 Pro → Accept agreements → Next → Install.
  17. Country/Region: Philippines.
  18. Keyboard Layout: US.
  19. Skip additional keyboard setup.
  20. Wait for setup.
  21. Device name: Admin-PC.
  22. Wait for configuration.
  23. Setup for Personal Use.
  24. Wait for downloading.
  25. Sign in with Microsoft Account → set PIN.
  26. Privacy Settings: turn off Location and Find My Device.
  27. Accept → wait → More options → Set up as new PC → wait.
  28. Skip → Skip → Skip → Decline → Decline → Skip for now.
