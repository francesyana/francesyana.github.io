Windows 10 Installation

This section provides a step-by-step demonstration of installing Windows 10. Watch the instructional video below to follow along with the process.

Resources

  • Windows 10 ISO: Download here

Windows 10 Installation Steps

  1. Create a new VM.
  2. Set Base Memory to 2048 MB.
  3. Assign 3 processors.
  4. Set Hard Disk size to 50 GB.
  5. Add ISO file.
  6. Start VM.
  7. Click Next → Install now.
  8. Select “I don’t have a product key” → choose Windows 10 Pro.
  9. Accept license agreement.
  10. Choose Custom settings → Next → wait for installation.
  11. Region: Philippines → Yes.
  12. Keyboard layout: US → Next → Skip.
  13. Setup for personal use.
  14. Select Offline account.
  15. Choose Limited experience.
  16. PC Name: Admin-PC.
  17. Create password.
  18. Set security questions.
  19. Select “Not now.”
  20. Accept terms.
  21. Skip remaining setup prompts.
