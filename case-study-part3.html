Case Study Part 3

This section presents Case Study Part 3: DHCP Server.

Steps

  1. DHCP Server Configuration
    • IP Block: 172.31.0.0 /16 and 10.0.0.0 /8
    • Router IP address must be a valid IP address
    • DNS server IP address is the same as the DHCP server. Also add Google (8.8.8.8, 8.8.4.4) and OpenDNS (208.67.222.222, 208.67.220.220) public DNS servers.
    • IP address lease expires every 8 hours
    • Do not distribute the first 30 and last 30 valid IP addresses
  2. Clone Windows Server 2008, Windows 7, and Windows XP VMs.
  3. Configure WS2008 network adapters:
    • Adapter 1 → Internal Network → rename as "XPFEMALENET"
    • Adapter 2 → Internal Network → rename as "7MALENET"
  4. Configure WSXP network adapter → rename Adapter 1 as "XPFEMALENET".
  5. Configure WS7 network adapter → rename Adapter 1 as "7MALENET".
  6. Start WS2008 and configure networking:
    • Control Panel → Network & Internet → Network Connections → Change Adapter Settings → LAN Properties → IPv4 Properties
    • (Female) Local Area Connection 1: IP 172.31.0.31, Subnet 255.255.255.0, Gateway 172.31.0.32, DNS 127.0.0.1
    • (Male) Local Area Connection 2: IP 10.0.0.31, Subnet 255.0.0.0, Gateway 10.0.0.32, DNS 127.0.0.1
  7. Add DHCP Role:
    • Start → Administrative Tools → Server Manager → Roles → Add Roles → DHCP Server
    • Validate DNS (127.0.0.1) → Alternate DNS: 8.8.8.8 → Install
  8. Configure DHCP Scopes:
    • Scope Name: Class A Female-XP → Range 172.31.0.1–172.31.0.254 → Subnet 255.255.255.0 → Exclusions: 1–30 and 225–254 → Lease: 8 hours → Gateway: 172.31.0.32 → Domain: tupt.edu.ph
    • Scope Name: Class B Male-7 → Range 10.0.0.1–10.0.0.254 → Subnet 255.0.0.0 → Exclusions: 1–30 and 225–254 → Lease: 8 hours → Gateway: 10.0.0.32 → Domain: tupt.edu.ph
  9. Configure DNS for scopes:
    • Remove 127.0.0.1
    • Primary DNS: 8.8.8.8
    • Secondary DNS: 8.8.4.4
    • Tertiary DNS: 208.67.222.222
    • Quaternary DNS: 208.67.220.220
  10. Configure Windows XP:
    • Rename computer to LASTNAME_CS-XP
    • Join domain: tupt.edu.ph
    • Authenticate with WS2008 Administrator credentials
    • Restart and log in via domain
  11. Configure Windows 7:
    • Rename computer to LASTNAME_CS-WIN7
    • Join domain: tupt.edu.ph
    • Authenticate with WS2008 Administrator credentials
    • Restart and log in via domain
  12. Verify DHCP leases:
    • WS2008 → Administrative Tools → DHCP → IPv4 → Address Leases
    • Check W7 lease in Class B Male scope
    • Check WXP lease in Class A Female scope
    • Run ipconfig /all in W7 and WXP to confirm DHCP assignment
