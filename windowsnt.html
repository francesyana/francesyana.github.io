Windows NT 4 Installation

This section provides a step-by-step demonstration of installing Windows NT 4. Watch the instructional video below to follow along with the process.

Steps

  1. Create a new VM.
  2. Set Base Memory to 256 MB.
  3. Add ISO: SP1.
  4. Add a Floppy Controller → then add floppy disk 1.
  5. In Settings → System → Acceleration → disable nested paging.
  6. Start VM.
  7. Enter → Enter → Continue → Page Down → F8 to agree → Enter → Enter → Format partition (NTFS) → Enter → wait → Enter → Enter → wait for setup to copy files → remove disk → Enter to restart → wait.
  8. After restart, system asks for disk → insert ISO file again (SP1).
  9. Click OK → Next → add disk again if missing → Typical → Next → Name: EndUser → Org: EndUser → Computer name: WIN NT 4 → Next → Password → Next → select NO → Next → Install → Next → Next.
  10. Networking: Next → Start Search → Next → leave TCP protocol checked → Next → Continue → Yes to DHCP → Next → Yes to Network → Name: NT 4 → OK → Next → Finish.
  11. Timezone: Beijing → Close → OK.
  12. Display: customize settings → Test → OK → Yes → OK → wait.
  13. Remove Disk/ISO → Restart.
  14. Choose 1 → wait → log in.
  15. Install Service Pack: add SP6 ISO → SP6 tab appears → close tab → go to Computer → select SP6 drive → Install SP6 → choose Intel-based system → Open → Accept → Install.
  16. After install, remove SP6 ISO → Restart → log in.
  17. Verify SP6: Computer → Properties or run winver.
  18. Add Guest Additions: Devices → Add Guest Additions → Next → Install → Finish → Restart → log in.
  19. Add SP1 ISO again → Start → Settings → Control Panel → Multimedia → Devices → Audio Devices → Add → Creative Labs Sound Blaster → OK → Continue → modify MPU I/O Address to disable → OK → Restart.
